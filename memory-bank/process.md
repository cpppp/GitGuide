**实现文件**：

- `core/history.py` - 历史记录管理
- `data/history.json` - 历史记录存储

**数据结构**：

```json
{
  "url": "https://github.com/user/repo",
  "name": "项目名称",
  "timestamp": "2026-03-18T18:30:00",
  "language": "Python"
}
```

**功能说明**：

- 保存最近 20 条分析记录
- 支持点击历史记录快速重新分析
- 支持一键清除所有历史记录

**验证方法**：
