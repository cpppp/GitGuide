## 1. 产品概述

**GitGuide** 是一个帮助开发者快速上手任意 GitHub 仓库的智能化工具。用户只需输入 GitHub 仓库 URL，系统即可自动分析仓库内容，生成一份定制的学习文档 + 启动指南，并结合 AI 能力提供交互式答疑，让用户可以边学边问，快速理解项目结构并成功在本地运行。

***

## 2. 目标用户

- 刚接触某个开源项目，希望快速了解的开发者。
- 需要接手同事或他人代码，希望快速熟悉项目结构的人。
- 学习编程，想通过实际项目练习但不知从何入手的学员。
- 技术团队负责人，需要快速评估开源项目可行性。
- 技术文档编写者，需要生成项目文档基础框架。

## 3. 核心功能

### 3.1 MVP 功能（v1.0 - 已实现）

| 功能         | 描述                                           | 状态    |
| :--------- | :------------------------------------------- | :---- |
| **URL 输入** | 用户在首页输入完整的 GitHub 仓库 URL（支持公开仓库）。            | ✅ 已实现 |
| **一键生成**   | 点击按钮后，系统开始分析仓库，生成学习文档和启动指南。                  | ✅ 已实现 |
| **学习文档**   | 包含项目概述、技术栈、目录结构说明、核心模块介绍、依赖项等。               | ✅ 已实现 |
| **启动指南**   | 提供分步骤的环境准备、安装命令、配置说明、运行命令，以及常见问题排查。          | ✅ 已实现 |
| **AI 问答**  | 在文档页面嵌入一个 AI 聊天助手，用户可针对当前项目提问，AI 结合仓库内容实时回答。 | ✅ 已实现 |

## 4. 用户流程

### 4.1 基础流程（MVP）

1. 用户访问 GitGuide 首页。
2. 在输入框中粘贴 GitHub 仓库 URL，点击"生成"按钮。
3. 系统展示加载状态，后台抓取仓库并进行分析。
4. 分析完成后，页面跳转至结果页，展示：
   - 左侧：学习文档（分章节）
   - 右侧：启动指南（步骤列表）
   - 底部或侧边：AI 聊天窗口
5. 用户可阅读文档，并随时向 AI 提问，AI 回答。

### 4.2 增强流程（迭代后）

1. **输入阶段**：
   - URL 格式实时验证
   - 历史记录快速选择
   - 收藏仓库快速访问
2. **分析阶段**：
   - 实时进度条显示
   - 分步骤状态提示
   - 预计剩余时间
   - 支持取消任务
3. **结果展示**：
   - 学习文档（分章节）
   - 启动指南（步骤列表）
   - 代码图谱可视化
   - AI 聊天窗口
4. **后续操作**：
   - 导出为多种格式
   - 分享分析结果
   - 收藏仓库
   - 提供反馈

- 支持私有仓库（通过 OAuth 授权）

- 增加"导出 PDF"功能
- 支持用户反馈，优化分析准确性

- API 文档自动生成
- 测试覆盖率分析
